welcome.

This is a blog where I talk about physics, namely: if you yourself had to rediscover the laws of physics, how would you do it? What experiments would you need to perform, what patterns would you need to notice?

As part of the discussion, I'll make a case for what I think the art of physics really entails. For an intro to my thoughts on this, check out the about page. Or, for a longer exposition, see my first post.
